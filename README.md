There are different directories in this repo, each one is for serving different purpose.

Fed: Performing federated learning exeperiments on the PC with flwr library.

Jetson_benchmark: This is for benchmarking different models in the jetson nano and calculate the resources and time taken by them.

Jetson_client_app: This is the client side app for jetson nano. It has a client code and server side code also.

Fed_jetson: This is a subset of client app. It is used for erforming federated learning experiment only, with jetson nano.

ALL THE DIRECTORIES HAVE THEIR CORRESPONDING README files.
